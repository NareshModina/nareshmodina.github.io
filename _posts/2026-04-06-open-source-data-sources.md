A curated list of open data sources for machine learning research and projects.

## Popular Open Data Repositories

- [OpenML](https://openml.org/){:target="_blank"} — collaborative platform with thousands of datasets and benchmarks
- [Kaggle](https://kaggle.com){:target="_blank"} — competitions and community-contributed datasets
- [Papers With Code](https://paperswithcode.com/datasets){:target="_blank"} — datasets linked directly to research papers and leaderboards
- [UC Irvine ML Repository](https://archive.ics.uci.edu/){:target="_blank"} — classic benchmark datasets
- [AWS Open Datasets](https://registry.opendata.aws/){:target="_blank"} — large-scale datasets hosted on S3
- [TensorFlow Datasets](https://www.tensorflow.org/datasets){:target="_blank"} — ready-to-use datasets via `tfds`
- [Hugging Face Datasets](https://huggingface.co/datasets){:target="_blank"} — the largest hub for ML datasets, strongly integrated with `transformers`

## Meta Portals

- [DataPortals.org](https://dataportals.org/){:target="_blank"} — directory of open data portals worldwide
- [OpenDataMonitor](https://opendatamonitor.eu){:target="_blank"} — European open data catalogue

## NLP

- [The Pile](https://huggingface.co/datasets/EleutherAI/pile){:target="_blank"} — 825GB diverse English text corpus by EleutherAI
- [Common Crawl](https://commoncrawl.org/){:target="_blank"} — petabyte-scale web crawl data, used to train most LLMs
- [C4](https://huggingface.co/datasets/allenai/c4){:target="_blank"} — cleaned Common Crawl, used to train T5
- [RedPajama](https://huggingface.co/datasets/togethercomputer/RedPajama-Data-1T){:target="_blank"} — 1T token open reproduction of LLaMA training data
- [ROOTS](https://huggingface.co/datasets/bigscience-data/roots_zh_wikipedia){:target="_blank"} — multilingual corpus used to train BLOOM
- [OpenSubtitles](https://opus.nlpl.eu/OpenSubtitles.php){:target="_blank"} — multilingual subtitle corpus, useful for dialogue and translation

## Computer Vision

- [ImageNet](https://www.image-net.org/){:target="_blank"} — the standard benchmark for image classification
- [COCO](https://cocodataset.org/){:target="_blank"} — object detection, segmentation, and captioning
- [Open Images](https://storage.googleapis.com/openimages/web/index.html){:target="_blank"} — 9M images with labels and bounding boxes by Google

## Tabular / Structured Data

- [UCI Repository](https://archive.ics.uci.edu/){:target="_blank"} — classic tabular benchmarks
- [Google Dataset Search](https://datasetsearch.research.google.com/){:target="_blank"} — search engine for datasets across the web
- [World Bank Open Data](https://data.worldbank.org/){:target="_blank"} — economic and development indicators
